# BooksApp

## 使用技術
### フロントエンド
- HTML
- CSS
- JavaScript(予定)
- Bootstrap(予定)

### バックエンド
- C# 7.3
- .NET Framework 4.7.2
- SQLServer 2019

### その他
- Visual Studio 2019
- Git/GitHub

## 機能一覧
### 本棚
- 登録されている本を一覧で表示
- 検索機能（実装予定）

### 本の追加
- 本棚に本を追加できます

### 本の詳細
- 登録している本の詳細を確認できます

### ブックマークの登録
- 本とページを指定してブックマークを登録できます（実装予定）
